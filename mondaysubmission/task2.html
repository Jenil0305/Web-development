INT 1010 HTML Programming Assignment

HTML is an important language used to create web pages. As a part of INT 1010, it is important that all students have experience in developing code.

Requirements

For this assignment, you are required to develop a 3-page website using Hypertext Markup Language (HTML). You need to choose what you would like to display on your website (the topic is up to you). The requirements for the assignment are as follows

Each page must contain the following:

  • Title, Heading, paragraph, image,link,backgroundcolor(Other than White)and text color(other than black)

In addition, the site must contain atleast:

  • 1 Horizontal rule
  • 1 external link

Grading Rubric
  • Site must have meaningful content relevant to site(10 points)
  • Site must have adhere to HTML guide and principles
    • --proper tag syntax--15
    • --appropriate number of pages--6
    • -links added and work--4
    • -Horizontal Rule-5
  • Site must be attractive(10 points)
    • Site is readable (--appropriate contrast for colors)
