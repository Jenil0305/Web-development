This assignment will give you practice creating HTML forms and using Javascript to process their contents to generate interesting or useful outputs. You will learn to:

  • Create simple HTML forms with input elements and a button;
  • Write Javascript functions that are used as event handlers,
  • Read into variables, perform some operations with them,
  • Write JQuery statements that gather the information from the input fields, store the values intovariables and perform some operations with them,
  • Display results to the user by dynamically changing the HTML page.
  • Incorporate conditional execution of code to perform file size calculations (Part 2),
  • Incorporate repetitive elements for multiple pages of a website by using JQuery load.
