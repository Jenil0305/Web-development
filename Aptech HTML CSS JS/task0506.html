What is Cricket all about?

  • It is a Bat and Ball game played between two teams of 11 player each.
  • The game is played in a field with a 22 yard long pitch in the centre of the field.
  • Cricket Ball,Cricket Bat,Stumps,Bails are some of the equipments used in Cricket
  • The game is played in three different format:
    • Test Match: A 5 Day game
    • One Day: Limited 50 over game for each side
    • T20: Limited 20 over game for each side

Interesting facts about Cricket

Here are some interesting facts about Cricket

  • Considered as world's 2nd most popular sport
  • The game is said to be originated in England
  • The first recorded game was played in 1646
  • Cricket is still not a part of Olympics
  • The game is popular in Australia,England,Indian Sub Continent,West Indies and Southern Africa
